Googleが発表したAIエディタ「Antigravity」を使えば、プログラミングの知識が全くない初心者でも、指示（プロンプト）を与えるだけでアプリを作成することが可能です。これは「バイブコーディング（雰囲気やニュアンスを伝えてコードを書かせる）」と呼ばれる新しい開発スタイルです。
以下に、初心者がAntigravityを使ってアプリを作る手順をステップ形式で解説します。
ステップ0：準備（インストールと設定）
アプリを作り始める前に、環境を整える必要があります。
1. Antigravityのインストール: 公式サイトからOS（Windows/Mac/Linux）に合ったインストーラーをダウンロードし、インストールします。
2. 日本語化（重要）: 初期状態は英語です。画面左側の「Extensions（拡張機能）」アイコンをクリックし、「Japanese Language Pack for Visual Studio Code」を検索してインストールし、再起動するとメニューが日本語になります。
3. ブラウザ拡張機能の導入: AIが作成したアプリの動作確認を自動で行えるようにするため、Google Chromeに「Antigravity Browser Extension」を追加します。これにより、AIがブラウザを操作してテストできるようになります。
4. AIの応答を日本語に固定: AIが英語で返答しないように設定します。エージェントパネルのメニューから「Customizations」→「Rules」→「Global」を開き、.mdファイルに「回答は常に日本語でお願いします」と記述して保存します。
ステップ1：プロジェクトの開始（Playgroundの活用）
初心者は、まず「Playground（プレイグラウンド）」という一時的な作業スペースを使うのがおすすめです。PC内のファイルを汚さずに実験できます。
1. エージェントマネージャーを開く: Ctrl + E（Macは Cmd + E）を押して、AI管理画面を開きます。
2. Playgroundを選択: 画面左側の「Start New Conversation」で「Playground」が選択されていることを確認します。
3. モデルの選択: 基本的には最新の「Gemini 3」や、賢いとされる「Opus 4.5」などを選択します。
4. モードの選択: 本格的なアプリを作る場合は、計画を立ててくれる**「Planning（プランニング）」モード**を選びます（簡単な修正なら「Fast」モードでも可）。
ステップ2：作りたいアプリを指示する（プロンプト入力）
チャットボックスに、作りたいアプリの内容を自然な言葉で入力します。これを「バイブコーディング」と呼びます。
• 入力例: 「クリスマスのカウントダウンをするアドベントカレンダーを作ってください。12月1日から25日までのカードを並べて、クリックすると画像が表示されるようにしてください」
• コツ: 詳細な仕様書を作るのが難しい場合、ChatGPTなどで事前に仕様書を作ってもらい、それをコピペして指示するのも有効です。また、参考画像や手書きのワイヤーフレーム画像を添付すると、AIがデザインを理解しやすくなります。
ステップ3：計画の確認と承認（Human-in-the-Loop）
指示を送ると、AIが自律的に「実装計画書（Implementation Plan）」と「タスクリスト」を作成します。
1. 計画のレビュー: AIが提示した計画書を開き、内容を確認します。もし要望と違う場合は、チャットで「ここはこう直して」と指示を追加します（例：「日付制御を追加して」など）。
2. 承認（Proceed）: 計画に問題がなければ「Proceed」ボタンを押して、開発をスタートさせます。
3. 作業の承認: 安全のため、AIがファイルの作成やコマンドを実行するたびに許可を求めてくる設定（推奨）になっています。「Accept」や「Proceed」を押して作業を進めさせます。
ステップ4：AIによる実装とテスト
承認後は、AIが自動でコードを書き、必要なライブラリをインストールし、ブラウザを起動して動作確認まで行います。
• 自動テスト: AIは作成したアプリが正しく動くか、ブラウザを操作してチェックし、そのスクリーンショットやログを「Walkthrough」というレポート（アーティファクト）として残してくれます。
• 画像の生成: Antigravityには画像生成AI（Nano-7など）も統合されているため、「このアプリに合う背景画像を作って」と指示すれば、画像を生成してアプリに組み込んでくれます。
ステップ5：完成と保存
1. 動作確認: 出来上がったアプリを自分で触ってみて、修正したい点があればチャットで追加指示を出します（例：「背景を宇宙っぽくして」など）。
2. 保存（Move to Folder）: Playgroundで作ったアプリが気に入ったら、画面右上のフォルダアイコンをクリックし、「Move Playground to a Folder」を選択して、PC内のフォルダに保存します。これで正式なプロジェクトとして保存されます。
補足：トラブル時の対応
AIがエラーを出して止まった場合や、修正したい箇所がある場合は、エラーメッセージを選択して「このエラーを直して」とチャットで指示すれば、AIが修正案を提示してくれます。
このように、Antigravityを使えば、コードを一切書かずに「指示→承認→確認」のサイクルだけでアプリを開発できます。
