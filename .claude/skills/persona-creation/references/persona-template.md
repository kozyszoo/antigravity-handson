# ペルソナプロファイル テンプレート

persona-writer はこのテンプレートに沿ってペルソナを執筆する。

## テンプレート

```markdown
# {ペルソナ名}（{セグメント名}）

> "{ペルソナの声を代表する一言}"

## 基本情報

| 項目 | 内容 |
|------|------|
| 名前 | {フルネーム} |
| 年齢 | {年齢} |
| 職業 | {職業・役職} |
| 家族構成 | {家族} |
| 居住地 | {都市名} |
| 年収 | {範囲} |

## ストーリー

{2〜3段落で、この人物の日常とプロダクトとの接点を物語的に描写}

## 行動パターン

- **デジタル利用**: {スマホ/PC/タブレットの利用状況}
- **情報収集**: {どこから情報を得るか}
- **購買行動**: {どう購入を決断するか}
- **1日のタイムライン**:
  - 朝: {行動}
  - 昼: {行動}
  - 夜: {行動}

## 心理特性

| 軸 | 内容 |
|----|------|
| 価値観 | {何を大切にするか} |
| モチベーション | {何がこの人を動かすか} |
| フラストレーション | {何にストレスを感じるか} |
| 恐れ | {何を避けたいか} |

## プロダクトとの関係

- **利用シーン**: {いつ・どこで・なぜ使うか}
- **ゴール**: {プロダクトで達成したいこと}
- **ペインポイント**: {現状の課題}
- **代替手段**: {今どうやって課題を解決しているか}
- **期待する体験**: {理想のUX}

## 影響要因

- **意思決定に影響する人**: {家族、友人、専門家など}
- **信頼する情報源**: {SNS、口コミ、レビューサイトなど}
- **ブランドへの態度**: {新しいもの好き/保守的}

## このペルソナへのアプローチ

- **マーケティングメッセージ**: {どんなメッセージが刺さるか}
- **チャネル**: {どこでリーチするか}
- **機能優先度**: {このペルソナが最も重視する機能}
```

## 執筆ガイドライン

1. **具体的な名前を使う**: 「30代女性」ではなく「田中美咲（34歳）」
2. **物語を添える**: データだけでなく、共感できるストーリーを書く
3. **リアリティを追求**: 年齢・職業・収入・行動の組み合わせが矛盾しないこと
4. **ステレオタイプを避ける**: 個人としての独自性を描く
5. **データに基づく**: user-researcher のリサーチ結果を反映する
6. **引用符の一言**: ペルソナの本質を捉える短いセリフを添える
