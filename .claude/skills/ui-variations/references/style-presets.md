# Style Presets Reference

5つのバリエーションに使用するスタイルプリセット候補。
リーダーはユーザーの要件に基づき、この中から5つを選択するか、カスタムスタイルを生成する。

## Preset Pool (12種)

### 1. Minimal Clean
- **コンセプト**: 余白と線の美学。必要最小限の要素
- **カラー**: 白/オフホワイト背景、黒テキスト、1アクセント色
- **フォント**: 細いウェイト (200-400)、大きな見出し
- **レイアウト**: 広い余白、薄いディバイダー、カードボーダーなし
- **タグ**: #minimal #whitespace #typography-driven

### 2. Bold Vibrant
- **コンセプト**: 強い色とコントラスト。目を引くビジュアル
- **カラー**: ビビッドなプライマリ + セカンダリ、白テキスト
- **フォント**: 太いウェイト (600-900)、大文字多用
- **レイアウト**: 大きなパディング、丸角ボタン、グラデーション背景
- **タグ**: #bold #colorful #gradient #high-contrast

### 3. Dark Mode
- **コンセプト**: ダークテーマ。夜間使用に最適化
- **カラー**: #0B-#15 背景、#F0 テキスト、ネオンアクセント
- **フォント**: 中ウェイト (400-600)、やや大きめの本文
- **レイアウト**: カード型、微妙なボーダー、グロー効果
- **タグ**: #dark #modern #glow #elevated

### 4. Organic Natural
- **コンセプト**: 自然・温かみ。手作り感のある親しみやすさ
- **カラー**: アースカラー（ベージュ、テラコッタ、オリーブ）
- **フォント**: セリフまたは手書き風、中ウェイト
- **レイアウト**: ソフトな角丸(20+px)、テクスチャ感、有機的な形状
- **タグ**: #organic #warm #earth-tones #friendly

### 5. Tech Futuristic
- **コンセプト**: テック・未来的。精密でシャープ
- **カラー**: ダーク背景 + ネオンブルー/パープル、モノスペースアクセント
- **フォント**: モノスペースまたはサンセリフ、等幅的
- **レイアウト**: グリッド重視、シャープな角(0-4px)、ボーダーアクセント
- **タグ**: #tech #futuristic #monospace #grid

### 6. Premium Luxury
- **コンセプト**: 高級感。洗練と上品さ
- **カラー**: ブラック/チャコール + ゴールド/シャンパン
- **フォント**: エレガントなセリフ、ライトウェイト
- **レイアウト**: 贅沢な余白、繊細なディバイダー、アニメーション想定
- **タグ**: #luxury #elegant #serif #gold

### 7. Playful Fun
- **コンセプト**: 遊び心。若いターゲット向け
- **カラー**: パステルまたはポップカラー、多色使い
- **フォント**: 丸ゴシック、バリアブルウェイト
- **レイアウト**: 非対称、丸い要素、イラスト想定スペース
- **タグ**: #playful #fun #pastel #rounded

### 8. Corporate Professional
- **コンセプト**: ビジネス・信頼感。プロフェッショナル
- **カラー**: ネイビー/グレー + ブルーアクセント
- **フォント**: クリーンなサンセリフ、安定したウェイト(400-600)
- **レイアウト**: グリッド整列、カード型、アイコン活用
- **タグ**: #professional #corporate #blue #trustworthy

### 9. Glassmorphism
- **コンセプト**: ガラス質感。透明感と奥行き
- **カラー**: 半透明白/グレー、グラデーション背景
- **フォント**: 軽いサンセリフ (300-500)
- **レイアウト**: ブラー背景カード、微妙なボーダー、レイヤー感
- **タグ**: #glass #blur #transparent #layered

### 10. Neubrutalism
- **コンセプト**: ネオブルータリズム。太い枠線と影
- **カラー**: 白背景 + ビビッドアクセント、黒ボーダー
- **フォント**: 太めのサンセリフまたはモノ (600+)
- **レイアウト**: 太いボーダー(2-4px)、ハードシャドウ、シャープ角
- **タグ**: #neubrutalism #bold-borders #hard-shadow

### 11. Soft Neumorphism
- **コンセプト**: ソフトな立体感。押し出し/凹み
- **カラー**: 同系色の微妙なグラデーション（#E0-#F5）
- **フォント**: 中ウェイトサンセリフ (400-500)
- **レイアウト**: ソフトシャドウ、凸凹効果、同トーン配色
- **タグ**: #neumorphism #soft #shadow #monochrome

### 12. Japanese Modern
- **コンセプト**: 和モダン。日本的ミニマリズム
- **カラー**: 和色（墨色、朱色、藍色、生成り）
- **フォント**: 日本語最適化、明朝体 or ゴシック体
- **レイアウト**: 縦書きアクセント、余白の美、非対称バランス
- **タグ**: #japanese #modern #wabi-sabi #asymmetric

## スタイル選択ガイドライン

リーダーは以下を考慮して5つを選択:
1. **差別化**: 5つが明確に異なるスタイル（似たもの同士を避ける）
2. **ターゲット適合**: ユーザーの指定した年齢層・用途に合うもの
3. **技術的実現性**: React Native / Web で実装可能なもの
4. **プラットフォーム**: iOS なら HIG 寄り、Web ならより自由に
