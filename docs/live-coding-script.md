# Google Antigravity ライブコーディング台本（デモ動画用）

この台本は、`docs/index.html`（ハンズオンガイド）の内容に沿って進めるライブコーディング動画の収録用です。

### 💡 撮影時のワンポイントアドバイス
* **待ち時間の処理**: 画像生成（Nano Banana）や `npm install` 等の待ち時間は、動画編集で速度を上げるかカットするとテンポが良くなります。
* **画面のズーム**: チャット入力時やプレビュー画面の表示時にズーム（パン＆ズーム）すると、視聴者が文字やUIを読みやすくなります。
* **事前の仕込み**: Step 4 の React プロジェクト構築は時間がかかる場合があるため、あらかじめ成功したバージョンのフォルダを準備しておき、動画では「ここまで進みました」と切り替えるのも一つの方法です。
* **エラー対応**: 万が一 AI がエラーを出した場合は、そのまま「エラーを修正して」と指示する様子を見せると、かえってリアリティが出てデモとして効果的です。
